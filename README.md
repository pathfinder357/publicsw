# Flask-Simple-Chat
Flask-socketIO를 이용한 심플한 채팅 어플리케이션 예제 코드



# Dependency & Version compatibility

Socket.IO 프로토콜은 서버와 클라이언트 간의 버전이 반드시 호환되어야만 동작합니다.
정상적인 실행을 위해, 반드시 requirements.txt에 기록된 버전으로 수행해주세요.



# GET STARTED

```shell
$ pip install -r requirement.txt
$ python chat.py
```
